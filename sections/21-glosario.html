Glosario de Modernización

Diccionario técnico y estratégico para la alineación del equipo IBS.

Estrategia y Gestión

Big Bang Migration
Sustitución total del sistema en un solo evento.
Ejemplo: "Apagar Sybase el viernes y prender .NET el lunes". Descartado por el alto riesgo de caída del servicio bancario.
Patrón Strangler Fig (Estrangulador)
Metodología inspirada en plantas que envuelven un árbol viejo hasta reemplazarlo.
Ejemplo: Crear una API en .NET para "Consultar Saldo" mientras el resto del Core sigue en PowerBuilder.
Wave (Oleada)
Conjunto de hitos que agrupan funcionalidades relacionadas para reducir dependencias.
Ejemplo: Wave 1 enfocado exclusivamente en canales digitales y consultas de clientes.

Arquitectura de Software

Anticorruption Layer (ACL)
Capa mediadora que traduce los modelos de datos del legado al nuevo diseño. Evita que la "suciedad" del código viejo contamine el nuevo sistema.
Bounded Context (Contexto Delimitado)
Frontera lógica donde un término tiene un único significado.
Ejemplo: El término "Cuenta" significa algo distinto para el módulo de Ahorros que para el de Contabilidad.
Monolito Modular
Arquitectura donde el código está unido en el despliegue pero separado físicamente en módulos independientes, permitiendo una migración futura a microservicios sin reescribir todo.

Datos, Integración y Resiliencia

ACID
Estándar de integridad: Atomicidad (Todo o nada), Consistencia (Reglas válidas), Aislamiento (Sin interferencias) y Durabilidad (Persistencia ante fallos).
Circuit Breaker (Disyuntor)
Patrón de resiliencia que detecta fallos en un servicio externo y detiene las llamadas temporalmente para evitar un colapso en cadena.
Idempotencia
Garantía de que procesar la misma petición dos veces no duplica el efecto.
Ejemplo: Si un cliente da click dos veces al botón de "Pagar", el sistema solo procesa un cobro.
SAGA Pattern
Gestión de transacciones largas entre módulos. Si un paso falla (ej. falla la contabilidad), se ejecutan Compensaciones para revertir pasos previos (ej. devolver el dinero al saldo).

Seguridad y Observabilidad

Correlation ID (Trace ID)
Identificador único que viaja con una transacción a través de todos los módulos. Permite rastrear un error desde la App móvil hasta la Base de Datos.
JWT (JSON Web Token)
Estándar seguro para representar la identidad del usuario en cada llamada a la API. Contiene "Claims" (permisos) firmados digitalmente.
Zero Trust (Confianza Cero)
Filosofía de seguridad donde no se confía en nadie por defecto, incluso si la petición viene de la red interna del banco.

Ecosistema Legado

Deuda Técnica
El costo acumulado por soluciones rápidas o parches del pasado que hoy dificultan la evolución del sistema.
Lógica "In-Database"
Reglas de negocio atrapadas en Stored Procedures (SPs). El objetivo de 2026 es extraer esta lógica hacia servicios C# testables.
Sybase ASE
Motor de base de datos relacional que sustenta el IBS actual. Su optimización es clave durante el periodo de Coexistencia.

Indicadores (KPIs)

MTTR (Mean Time To Repair)
Tiempo promedio en que el equipo resuelve un fallo crítico. Mide la agilidad de respuesta ante crisis.
P95 Latency
Métrica de velocidad: indica que el 95% de los usuarios experimentan tiempos de respuesta menores a X milisegundos. Es más real que el "promedio".
